# fsr2022

For launch simulation 

```
roslaunch iris_model spawn_hummingbird.launch 
```


For launch controller  and APF
```
rosrun geometric_controller traj
rosrun geometric_controller geometric_controller_node
```


Requirement:
```
-ROS-Noetic
-Gazebo
-mav_msgs
-gazebo_msgs
-sensor_msgs
```
